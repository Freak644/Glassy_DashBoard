# Glassy Dashboard 🚀

Modern glass-style Chrome new tab extension with:
- Live clock
- Favorite shortcuts
- Custom background
- Local storage support

## Installation

1. Download the ZIP from Releases
2. Extract it
3. Go to chrome://extensions
4. Enable Developer Mode
5. Click Load Unpacked
6. Select the extracted folder

Enjoy 🔥
